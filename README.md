# MATLAB Class A Amplifier

This university project presents a MATLAB GUI developed for analyzing and visualizing the behavior of a **Class A amplifier**.

## Project Overview
The application was created as part of a semester project for **Fundamentals of Computer Aided Graphics**. It combines theoretical calculations with a graphical user interface in MATLAB to help visualize amplifier-related signals and parameters.

## Main Features
- MATLAB GUI for Class A amplifier analysis
- Adjustable input values for **Vsat** and **Vcc**
- Plots for:
  - input voltage (Vi)
  - output voltage (Vo)
  - collector-emitter voltage (Vce)
  - output current (Io)
  - collector current (Ic)
  - load line (RLOAD)
- Buttons for circuit view, documentation, plotting, and visibility control
- GUI elements such as popup menu, axes, edit boxes, panels, static texts, and push buttons
